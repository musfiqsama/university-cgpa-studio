# 🎓 University CGPA Studio

A modern and responsive CGPA calculator with dashboard, planner, and report features. Built to help students easily track, plan, and manage their academic performance.

---



## 🚀 Features

* 📊 **CGPA Calculator** – Calculate GPA/CGPA easily
* 📅 **Semester Planner** – Plan subjects and credits
* 📈 **Dashboard** – Overview of academic performance
* 🧾 **Report System** – Clean report view (print/PDF ready)
* 💾 **Auto Save** – Data stored locally in browser
* 📱 **Responsive Design** – Works on mobile, tablet, and desktop
* 🎨 **Clean UI** – Modern and user-friendly interface

---

## 🛠️ Tech Stack

* ⚛️ React
* ⚡ Vite
* 🎨 Tailwind CSS
* 📊 Recharts
* 🎬 Framer Motion

## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/musfiqsama/university-cgpa-studio.git
cd university-cgpa-studio
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
